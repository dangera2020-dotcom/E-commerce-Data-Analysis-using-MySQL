# E-commerce-Data-Analysis-using-MySQL

**Project Overview**
This project analyzes an e-commerce dataset to generate business insights related to sales, customer behavior, delivery performance, and seller performance.
The analysis is performed using MySQL, focusing on joins, aggregations, CTEs, and window functions.

**Dataset**
The dataset contains multiple tables related to an e-commerce platform:
-Orders
-Customers
-Order Items
-Products
-Sellers
-Payments
-Reviews
-Geolocation

**Key Analysis Performed**
-Sales Analysis
-Customer Analysis
-Delivery Analysis
-Review Analysis
-Payment Analysis
-Seller Analysis

**Key Insights**
Most revenue is generated from top product categories
Credit card is the most used payment method
Late deliveries negatively impact customer ratings
A small number of sellers contribute to a large portion of revenue
Delivery time varies significantly by region
